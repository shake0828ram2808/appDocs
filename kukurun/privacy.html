kukurunについて

「kukurun（くくるん）」は、6〜8歳のお子さまを対象とした九九学習アプリです。開発者は、自身の6歳の子が九九に興味を持ったことをきっかけに、楽しく学べるアプリ開発に取り組みました。

繰り返し練習しながらメダルを集めることで、お子さまが楽しく・自然に九九を身につけられることを目指しています。

動作環境

  • ✓ スマートフォン・タブレット・PC のブラウザで動作します。
  • ✓ 音声読み上げ機能は、Web Speech API に対応したブラウザ（Chrome・Safari・Edge など）でお使いいただけます。
  • ✓ インターネット接続は初回読み込み時のみ必要です。

学習データについて

  • ✓ 学習の進捗・獲得メダルは、お使いの端末のローカルストレージにのみ保存されます。
  • ✓ データはブラウザの設定またはキャッシュ削除で消去できます。
※ データは端末に紐づいています。機種変更・ブラウザ変更の際は引き継ぎができませんのでご了承ください。

個人情報について

  • ✓ 本アプリは、氏名・メールアドレス・住所などの個人情報を一切収集しません。
  • ✓ アカウント登録やログインは不要です。
  • ✓ お子さまの学習情報や行動履歴を外部サーバーへ送信することはありません。

音声機能について

  • ✓ 本アプリの音声読み上げは、端末に内蔵された音声合成機能（Web Speech API）を使用しています。
  • ✓ 音声データは端末内のみで処理され、外部へ送信されることはありません。
  • ✓ マイクなど、音声を「録音」する機能は使用していません。

データの保存について

  • ✓ 学習の進捗・獲得メダルなどのデータは、お使いの端末のローカルストレージにのみ保存されます。
  • ✓ これらのデータが外部のサーバーや第三者に送信されることはありません。
  • ✓ データはブラウザの設定から削除できます。アプリのアンインストールやキャッシュ削除によっても消去されます。
※ データは端末に紐づいています。機種変更・ブラウザ変更の際は引き継ぎができませんのでご了承ください。

広告・外部サービスについて

  • ✓ 本アプリには広告は表示されません。
  • ✓ Google Analytics・Firebase などのアクセス解析ツールは使用していません。
  • ✓ サードパーティ製のSDK・トラッキングライブラリは一切組み込まれていません。

ポリシーの変更について

本プライバシーポリシーは、必要に応じて更新することがあります。 変更があった場合は、このページに掲載します。
