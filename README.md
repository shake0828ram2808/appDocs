# appDocs

アプリごとのプライバシーポリシー等を管理するリポジトリです。  
GitHub Pages で配信します。

## URL構成

```
https://shake0828ram2808.github.io/appDocs/<appname>/privacy.html
```

## フォルダ構成

```
/
├── index.html
├── kukurun/
│   └── privacy.html
└── <appname>/        ← 新しいアプリはここに追加
    └── privacy.html
```

## GitHub Pages の有効化

1. リポジトリの **Settings > Pages** を開く
2. Source を **Deploy from a branch** に設定
3. Branch を **main** / `/ (root)` に設定して保存
